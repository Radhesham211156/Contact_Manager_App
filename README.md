# ⚡ Smart Contact Manager

A modern Full Stack Contact Management Application built using **Java**, **Spring Boot**, **MySQL**, **HTML**, **CSS**, and **JavaScript**.

The application provides secure and efficient contact management with RESTful APIs, responsive UI, and clean backend architecture.

---

## 🚀 Features

* Add New Contacts
* Update Existing Contacts
* Delete Contacts
* View Contact Details
* Manage Favorite Contacts

---

## 🧰 Tech Stack

### Frontend

* HTML5
* CSS3
* JavaScript
* Bootstrap

### Backend

* Java
* Spring Boot
* Spring Data JPA
* Hibernate

### Database

* MySQL
